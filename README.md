# GG-Web

这是从当前项目额外整理出的 GitHub 上传版副本，不影响原始目录内容。

## 目录说明

- `src/` React 页面与组件源码
- `public/` 静态资源
- `index.html` 页面入口
- `package.json` / `package-lock.json` 项目依赖与脚本
- `vite.config.js` Vite 配置

## 本地运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 说明

- 这个目录适合直接作为 GitHub 仓库内容上传
- 未包含 `node_modules/`、`dist/`、本地日志、临时截图等非源码文件
